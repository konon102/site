Safing.io

Safing.io — это комплексный инструмент для защиты конфиденциальности в интернете, который предоставляет несколько уровней безопасности, включая блокировку отслеживающих сервисов, управление сетевыми соединениями и анонимность для пользователя.

Преимущества

  • Многоуровневая защита: Обеспечивает защиту от множества видов угроз и атак.
  • Поддержка анонимности: Использует технологии для скрытия личных данных и местоположения.
  • Интуитивный интерфейс: Удобный интерфейс, подходящий как для новичков, так и для опытных пользователей.

Недостатки

  • Платность: В отличие от некоторых аналогичных решений, часть функционала доступна только в платной версии.
  • Зависимость от сервера: Программа может зависеть от работы внешнего сервера для некоторых функций.
  • Проблемы с совместимостью: Могут возникать проблемы с совместимостью на некоторых платформах и в специфических сетевых конфигурациях.

1. Раздел "Network Activity"

Этот раздел отображает текущую и историческую сетевую активность приложений.

Компоненты:

  • Заголовок "Network Activity" — указывает, что раздел посвящён мониторингу сетевых подключений.
  • Примечание: "No network history data available" означает, что история не записывается или не включена.
  • Поиск и фильтрация — пользователь может искать подключения по ключевым словам или фильтровать данные за последние 10 минут. Если включена история, можно анализировать более старые данные.
  • Таблица "Search for connection types" содержит столбцы Aborted, Predicted, Quantity/Demand/Age, Notes/Change By/Staff. Таблица пуста, что может означать отсутствие активных подключений или ошибку загрузки.
  • Список "CONNECTIONS" — перечислены типы подключений (например, HTTP, DNS).
  • Результаты подключений (#17 Results) — список активных или недавних соединений, включая домены и приложения. Кнопка "Previous" возвращает к предыдущим результатам.

Для чего нужен раздел?

  • Контроль трафика: какие приложения куда подключаются.
  • Выявление подозрительных подключений (например, к рекламным доменам).
  • Настройка правил блокировки/разрешения.

Для чего нужен раздел?

  • Поддержка: связь с разработчиками.
  • Улучшение продукта: отправка отзывов и багов.

3. Раздел "All Apps" (safquad.png)

Здесь отображаются все приложения, взаимодействующие с сетью, сгруппированные по категориям.

Компоненты:

  • Группы приложений: Browser, Background, Media, Network, Update, System, Other, Not assigned.
  • Кнопка "Export Settings": экспорт текущих настроек безопасности в JSON.
  • Приложения внутри групп: каждая программа с иконкой и названием.

Для чего нужен раздел?

  • Обзор всех программ с сетевым доступом.
  • Управление правилами безопасности для каждой программы.
  • Экспорт/импорт настроек для резервного копирования или переноса.

4. Раздел "Dashboard"

Это главная панель управления, которая предоставляет обзор текущего состояния защиты и сетевой активности.

Компоненты:

  • Приветствие и статус: "Welcome back!" — приветствие пользователя.
  • Уведомление о новых плагинах или обновлениях: "4 New CDs plugin's open dashboard".
  • Блок "FULLNESS": Перечислены модули и их статусы:
    • Secure DNS — защищённый DNS (активен, бесплатен).
    • Privacy Filter — фильтр приватности (активен, бесплатен).
    • Network History — история сетевой активности (доступна в платной версии Permanent Plus).
    • Safing Support и Safing Privacy Network — дополнительные функции (требуют подписки).
  • Новость о партнёрстве с VPN: "Safing Joint Forces with VPN".
  • Блок "Dataflow": Информация о сетевых обновлениях и списке служб и приложений.
  • Статистика "RECENT ACTIVITY": Таблицы с данными по подключениям, активным приложениям, объёмам данных и статистике.
  • Блок "ACTING BLOCKED CONNECTIONS": Таблица с заблокированными подключениями по странам (Россия, Италия, США и др.).
  • Уведомление о текущем тарифе: "Your current plan is Permanent Free and cable stops!"
  • Раздел "NEWS": Повторение новости о сотрудничестве с VPN и ссылки на домены, связанные с рекламой или отслеживанием.

Для чего нужен раздел?

  • Обзор состояния защиты в реальном времени.
  • Контроль сетевой активности и заблокированных подключений.
  • Уведомления о новых функциях и обновлениях.

5. Раздел "Secure" (Settings)

Этот раздел посвящён настройкам безопасности и управления подключениями.

Компоненты:

  • Уведомления: Например, "Which Name is in 4" — уведомление о нераспознанном процессе.
  • Список процессов: "Windows License Manager", "Open OS Internet Browser", "Unreal Engine", "Microsoft® Windows® Operating...", "Microsoft OneDrive".
  • Настройки браузера: "Open GX Internet Browser", пути к файлам и опции ("Block Connection", "Keep history on File", "User SPI on File").
  • Кнопки: Block Connection, Settings, Details, Insights.
  • Privacy Filter: Настройки фильтра приватности ("Internet Scope", "Connection Type", "Delete", "Print List").
  • Default Network Action: Действия по умолчанию ("Force Block Internet Access", "Force Block LAN").
  • Ошибка: "AccruBause Windows" — сообщение о проблеме с Windows.

Для чего нужен раздел?

  • Управление безопасностью отдельных приложений.
  • Настройка правил блокировки и фильтрации трафика.
  • Диагностика проблем с подключениями.

Детальная настройка профилей приложений и отладка.

Компоненты:

Для чего нужен раздел?

  • Тонкая настройка поведения приложений.
  • Контроль над действиями конкретных процессов.
  • Отладка и журналирование активности программ.

7. Раздел "Secure" (Connections)

Общая информация о браузере

Print: Путь к исполняемому файлу (C:\Users\Jason\AppData\Local\Programs\Opera GX).

Binary: Имя процесса (open.exe).

Active Connections: Количество активных подключений (218).

Network History: Отсутствует (None).

Для чего нужно? Быстрый доступ к расположению браузера и мониторинг его сетевой активности.

Блокировка подключений ("Block Connections")

Key History: Функция доступна в платной версии Get Plus (возможно, журналирование ключевых событий).

Use SPN / SPN Exit: Требуют подписки Get Pro (SPN — Safing Privacy Network, защищённый VPN-выход).

More: Дополнительные опции.

Что будет при нажатии?

  • Платные функции перенаправят на страницу подписки.
  • "More" — раскроет расширенные настройки блокировки.

Управление подключениями ("Connections")

Кнопки:

  • Settings: Настройки фильтрации трафика.
  • Details: Техническая информация (IP, порт, протокол).
  • Insights: Аналитика трафика (геолокация, частота запросов).
  • Поиск: Поле для фильтрации подключений (нажать EXITER для применения).

Для чего нужно? Ручное управление каждым соединением (например, блокировка рекламных доменов).

Фильтры ("Aboved")

Provider: Провайдер (например, Cloudflare).

Country: Страна сервера.

Domain: Домен подключения.

More: Дополнительные критерии (например, тип трафика).

Пример использования: Блокировка всех подключений к серверам в определённой стране.

Группировка и сортировка ("Connection")

Group By: Группировка по доменам, странам и т.д.

Sort: Сортировка по времени, количеству запросов.

Для чего нужно? Удобный анализ трафика (например, выявление самых активных доменов).

Автообновление ("Reload")

Интервалы обновления данных: от Gx min_auto (настраиваемый интервал) до < 1 min_auto (каждую минуту).

Что делает? Позволяет настроить частоту обновления списка подключений.

Таблица активных подключений ("AGA Results")

Отображает детали последних соединений:

Колонка Описание Пример значения
Domain/IP Адрес сервера features.opera-apt2.com (82.145.215.99:443)
Status Статус (начато/завершено) started / ended
Time Время активности 1 min ago
Action Правило (например, блокировка) Default Network Action
Scope Уровень применения Global (для всех процессов)

Примеры строк:

  • Подключения к traffink.syz (возможно, VPN-сервис) с повторяющимися запросами.
  • Домен opera-apt2.com — вероятно, серверы обновлений Opera GX.

Для чего нужно?

  • Выявление подозрительных подключений (например, неизвестные VPN).
  • Проверка, какие ресурсы потребляют трафик.

Ключевые функции раздела

  • Блокировка в один клик: Запрет подключений к выбранным доменам.
  • Глубокая аналитика: Просмотр геолокации серверов и типов трафика.
  • Гибкая настройка: Группировка, сортировка, интервалы обновления.
  • Интеграция с SPN: Использование Safing Privacy Network для анонимизации трафика (в Pro-версии).

Что произойдёт, если...

  • Заблокировать домен (например, traffink.syz)? Браузер не сможет соединяться с этим сервером — это может нарушить работу некоторых функций (например, VPN).
  • Включить "Use SPN"? Трафик Opera GX будет перенаправляться через защищённую сеть Safing (если есть подписка).
  • Нажать "Details"? Откроется техническая информация: TLS-версия, сертификаты, продолжительность сеанса.
